Como se siente

Un tablero vivo, aunque no tenga base de datos.

La pagina esta pensada como una vitrina clara para reportes del campus. Por ahora es estatica, pero queda lista para conectar datos cuando quieras darle vida con JavaScript.

Detalle humano

No es solo una lista de objetos.

Es una pequena memoria del campus: una botella olvidada despues de clase, unas llaves que detienen todo el dia, un cuaderno lleno de apuntes antes de un examen. La interfaz intenta tratar esos reportes con cuidado, orden y un poco de caracter.

Lo Encontre Hecho para recuperar historias, no solo cosas.
